Sobre o projeto

O projeto IndaiáCultural é o resultado do primeiro projeto da disciplina de Desenvolvimento Web do Instituto Federal de Educação, Ciência e Tecnologia de São Paulo - Campus Hortolândia, o projeto tem o intuito de disponibilizar para rede uma página que demonstra os pontos culturais de uma cidade escolhida pelos alunos, a cidade escolhida neste caso foi a de Indaiatuba, a página permite a visualização dos principais pontos culturais e históricos da cidade de Indaiatuba e apresenta informações básicas como telefone, horários de disponibilidade e uma imagem da localidade.

Utilizando recursos como JavaScript, HTML e CSS foi realizada a criação desta página que disponibiliza informações sobre os pontos culturais da cidade de Indaiatuba - SP. O projeto foi realizado pelo aluno Nathan Gustavo Vieira Ribeiro do curso de Análise e Desenvolvimento de Sistemas (5º Semestre), para a disciplina de Desenvolvimento Web.
